Loxone Demo Case · Quick Start Guide

Get started with Loxone in 30 minutes or less

Thank you for purchasing the Loxone Demo Case and welcome to Loxone!

The Demo Case is your introduction to the basics of Loxone hardware and software.
By the end of this guide you will have successfully:

  • Connected and powered on your Loxone Demo Case
  • Tested the basic functionality of the Touch Tree, Presence Sensor Tree, and the RGBW 24V Dimmer Tree
  • Connected to the Loxone Demo Case with the Loxone App and Loxone Config
  • Understood how to control the Miniserver and Tree elements manually, with the Loxone App, and with Loxone Config

Inside the case you'll find a Miniserver (the brain that runs every automation), a wired Tree extension for wired devices, an Air Base extension for wireless devices, an RGBW 24V Dimmer Tree extension, and a WiFi router so your phone or PC can talk to it.

⏱ ~30 minutes total ✓ No prior experience needed

What comes in the Demo Case?

Part 1 of 3

Power on and test ~10 min

Get the Demo Case up and running and start using the Tree devices right away, before touching any software.

1

Connect

Plug in the power

Connect the supplied power adapter to the socket on the back of the case, then into a wall outlet.

2

Verify

Wait for green lights

The Miniserver boots automatically. The case is ready when all LEDs blink green, which takes just a few seconds.

3

Ready

Everything is already configured

The default configuration allows you to manually operate the built in devices right away. No setup is required.

The Miniserver comes pre-loaded with the standard configuration. The Touch Tree operates according to the Loxone Switch Standard. The Presence Sensor Tree responds to movement according to the Lighting Controller moods. The toggle switches turn the LED lights on and off, and the potentiometer activates and controls the rate of the pulse generater for the digital output light sequencer.

Try it yourself

Exercise 1

Turn on the LED light strip

Goal
See how the Touch and RGBW Trees function.
  1. Touch the center of the Touch Tree listen for the click sound and watch the LED strip light up red.
  2. Touch the center of the Touch Tree again to toggle through the remaining 3 lighting moods.
  3. Single or double touch the Touch Tree to switch off the LED strip.
The center of the Touch Tree switch turns the connected LED strip on, toggles through the pre-configured lighting moods, and then turns the LED strip off. A double touch of the Touch Tree will turn off the LED strip regardless of which mood is currently selected. The Miniserver comes configured with a Lighting Controller function block that includes 4 lighting moods (Red, Green, Blue, Bright) followed by Off.
Exercise 2

Toggle the digital output lights on and off

Goal
See how digital inputs control digital outputs.
  1. Flip one of the 7 toggle switches connected to the digital inputs and watch its LED turn on.
  2. The toggle switches function as both a momentary (down position) and latch (up position) switch.
  3. Close the window contact sensor to switch on the Q8 digital output light.
Each digital input toggle switch is wired to a digital output. Flipping the switch tells the Miniserver to drive digital output LED according to the configuration. This demonstrates the potential functionality of digital switch inputs and outputs.
✓

What you've learned: the basics of Touch and RGBW Tree functionality, how digital and analogue inputs (switches, potentiometers, sensors) can drive digital outputs (LEDs) through the Miniserver.

Part 2 of 3

Control Miniserver using the Loxone App ~10 min

Connect to the Demo Case Miniserver from your mobile device or browser using the Loxone App to control it remotely.

1

Install

Install the Loxone app

Download and install the free Loxone App for your mobile device.

2

Connect

Join the Demo Case WiFi

On your phone or PC, find the WiFi network named after the case serial number (it starts with 504F94…), printed on the white sticker inside the case. Connect to it.

3

Control

Use the Loxone App on mobile device or browser

In the Loxone App on your mobile device click Add Miniserver and then click Search to find your Demo Case and connect using admin / admin as the username and password. Or click Add Manually and type the S/N of the Miniserver (same as WiFi network) click Connect to connect to the Demo Case Miniserver.

Or, in any browser, go to http://MS + the last 4 digits of the serial number (e.g. http://MS00FF/) and log in with admin / admin.

Important
The default login is admin / admin. We recommend that you change it after you log in for the first time..
The Loxone App gives you a clean, user friendly visualization of the Miniserver settings and controls. It mirrors the Miniserver in real time and also sends you alerts with notifications. Every on-screen control maps to a real device, and updates live as things change. The Demo Case room in the Loxone App let's you control the LED strip lighting by selecting a pre-configured mood or manually adjusting the color and brightness.

Try it yourself

Exercise 3

Toggle a light from the app

Goal
Control LED strip with live visualization in app.
  1. Make sure you phone is connected to the Demo Case WiFi.
  2. Open the Loxone App and navigate to the Rooms screen and select the Demo Case room.
  3. Turn on the LED strip with the switch in the Loxone App and watch the LED strip light up.
  4. Select a different mood and customize the color and brightness of the LED strip.
Exercise 4

Do the same from the browser

Goal
Control from any device without installation.
  1. Make sure you computer is connected to the Demo Case WiFi.
  2. Open any browser and navigate to http://MS + last 4 digits of S/N.
  3. Repeat steps from Exercise 4.
✓

What you've learned: the same Miniserver is reachable from the Loxone App on a mobile device and in any browser. You can control the miniserver live in a clean and user friendly control panel.

Part 3 of 3

Make it your own in Loxone Config ~10–15 min

Connect to the Miniserver with the Loxone Config , load the configuration, change something, test it, and save it back.

1

Install

Download Loxone Config

Download Loxone Config, the free configuration software for Windows, and install it on your PC.

2

Connect

Search & connect to the Miniserver

Connect to the Demo Case WiFi on your PC, just as you did in the previous part. Open Loxone Config and click on Search local network... under the Miniserver section of the left sidebar navigation. Click on the Demo Case and use admin / admin as the user name and password. Then click Connect to log in.



The top left status bar turns green and says Connected when the Miniserver is connected with Loxone Config.


3

Load

Load the program from the Miniserver

The pre-configured program should automatically load from the Miniserver. If it does not, simply click the Load program from Miniserver download button to load the program stored on the Miniserver.



Once you’ve tested and made any changes to your configuration, you can upload it onto your Demo Case Miniserver, by clicking the ‘Save program in Miniserver’ upload button.

Tip
Before changing anything, we recommend that you save a local backup so you can always restore the factory program. (You can download the factory program HERE.)

For further information on using Loxone Config have a look at our documentation pages “Auto Configuration” and “How to create logic in Loxone Config”.

Try it yourself

Exercise 5

View live interactions

Goal
See how the Miniserver and Loxone Config interact in real time.
  1. Once your Miniserver is connected to your Loxone Config, click on the Demo Case Miniserver icon in the Project Tree.
  2. In the Demo Case page you should see a visualization of the Miniserver.
  3. Click the Start LiveView with manual value adjustment button.
  4. Toggle the digital input switches in the Demo Case and notice how the visualization changes to match the current settings.
✓

What you've learned: you can connect your Miniserver to Loxone Config, load the program, make changes with LiveView enabled, and upload any changes back to the Miniserver. This is the core workflow of Loxone Config.
