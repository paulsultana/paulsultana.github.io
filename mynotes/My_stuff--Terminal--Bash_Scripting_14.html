Filtering IPs with Ping Response

Script:

#!/bin/bash

for ip in `seq 1 254`; do

ping -c 1 $1.$ip | grep "64 bytes" | cut -d " " -f 4 | tr -d ":" &

done

the for loop creates a variable (ip) and then loops through a sequence 255 times (for full IP range)

then for each IP it pings one time based on the user input (first part of IP 111.111.111.) and searches for line starting with “64 bytes"

the results are cut up by spaces delimiter and the 4th field is returned, then the colon is deleted

*can replace & with ; to make output list in order not randomly

*before executing script you need to change its mode to executable using chmod +x ipsweep.sh then it will change to green when you ls

*to execute input ./ipsweep.sh 111.111.111 in terminal

*to write the output to a file use ./file.name 111.111.111 > iplist.txt

Loop Through Results

In-line Script:

for ip in $(cat iplist.txt); do sudo nmap -sS -p 80 -T4 $ip & done

the for loop creates a variable (ip) and loops through list of IP addresses

then for each it does nmap stealth scan of port 80 with speed of t4

$ip declares the IP address (in nmap call)

& runs multiple in parallel (also outputs at random - faster but harder to read output) - can replace with ;
