Secondary DNS


To test if a domain is setup with WEDOS secondary DNS you can:

dig @{DNS IP} domain.com SOA

By digging as a DNS IP you can see if the following is true, and therefore is setup as secondary (or primary):

  • status: NOERROR
  • flags: aa

This means that the DNS IP you used is authoritative and can serve the DNS records for that domain.

To get the DNS IP just dig the NS (ns.wedos.com -> 45.138.106.0).

***both the primary and secondary DNS will look the same - so you need to know which one is primary by digging the domain first***

If status returns as REFUSED or SERVFAIL, or there is not aa flag, then the DNS server is not serving the zone.
