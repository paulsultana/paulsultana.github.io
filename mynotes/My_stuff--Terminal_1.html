Keyboard shortcuts:

ctrl + arrow left/right - jumps quickly through command

ctrl + a - jumps to the beginning of the command

ctrl + e - jumps to the end of the command

ctrl + d - logs out

ctrl + l - cleans terminal (by moving down and starting a new line - not deleting everything)

ctrl + u - cuts everything from the cursor to the beginning of the line

ctrl + k - cuts everything from the cursor to the end of the line

ctrl + w - deletes the word immediately before the cursor

ctrl + y - pastes the last cut text
