Command Line: Navigating File System

*tab to autocomplete

clear - clear terminal

code . - opens VScode

xdg-open file.name - opens a file in the default app

pwd - present working directory - where am I now?

cd - change directory - .. goes back one

ls - list directory contents - use -a or -la to view hidden files/folders

*to cd or ls directly from a different directory, use /root/directoryname/ or ~/directoryname/ and hit tab twice

mkdir - make directory

rmdir - remove directory

cp - copy file - cp filename.txt Newlocation/

rm - remove file - rm Newlocation/filename.txt (or just file name if in current dir)

mv - move file - mv file.name Newlocation/

locate - find a file in system

man - manual page - man ls (man followed by command - or command --help)

echo “string” - echos/prints a string to Terminal (or to a file with echo “string” > file.txt)

append text with echo “new text to be appended” >> file.txt

cat file.txt - prints the contents of a file to Terminal

touch newfile.txt - creates a new file

nano - a text editor within Terminal (nano newfile.txt opens the file editor and create the file)

gedit - opens a graphic interface text editor from within Terminal (gedit newfile.txt opens a file)

grep “text” - searches output for specified text and only returns those lines

cut -d “ ” -f 4 - cuts up the output based on the delimiter (-d) and then return the specified field (-f)

tr -d “:” - translate transforms or deletes (-d) the specified characters from the input

Permission/Users

chmod +/-(rwx) or 777 - change mode of file/directory - use +/- and/remove r and/or w and/or x to change permissions (777 = +rwx)

adduse name - adds a user with one or two names

cat /etc/passwd - shows a list of users on system

cat /etc/shadow - shows hash of passwords on system

su name - switch user (use root for root)

passwrd username - changes the password for the user (can use sudo before)

Network Services

python3 -m http.server - will spin up a simple server from the active directory (0.0.0.0 on port 8000)

can access with 0.0.0.0:8000, or your IP addr:8000, or localhost:8000

php -S localhost:8000 - will spin up a php development server from the active directory (http://localhost:8000/filename)
