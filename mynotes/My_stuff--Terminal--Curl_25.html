Curl Commands

Timing website response: (use shortcut bash ttfb.sh)

curl -o /dev/null -s -w "TTFB: %{time_starttransfer}\n" https://paulsultana.com

Explanation:

-o /dev/null → discard the output

-s → silent mode

-w → custom output, showing only the time until first byte

Example output:

TTFB: 0.245

This means 245 ms to first byte.

curl -I --resolve www.auma.de:443:185.8.237.6 https://www.auma.de

This forces curl to connect to one specific proxy node.

Explanation:

--resolve hostname:port:IP

So curl will:

connect to 185.8.237.5

send Host: www.auma.de

use TLS SNI www.auma.de

That simulates a real client request but lets you choose the proxy node.

curl ifconfig.me - use for my public ip

curl - loads a url and returns the content (-L follows redirects, -I fetches headers only)

curl ifconfig.me - my public IP address

curl 'url' | jq '.' - for beautifying JSON

curl -x HTTP method URL - allows you to specify which HTTP method (GET, PUT, POST, DELETE) - without -x it defaults to GET

curl -I -H "User-Agent: Mozilla/5.0" -H "Accept: */*" https://wedospresales.fun
